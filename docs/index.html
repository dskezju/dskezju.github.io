知识工程与知识系统团队


实验室致力于人工智能（AI）与基于知识的系统（KBS）应用于复杂系统设计、分析、制造和维护，近年来主要从事知识图谱构建、基于知识的决策与推理、知识驱动的协同建模与仿真、复杂系统状态监测和故障诊断等方面的研究工作
